HTML Structure

This template uses the unsemantic 1170 grid system. If you're unfamiliar with the unsemantic 1770 grid system, be sure to read about it here.

All grid units are contained within a top level element, with the class name w-container. This has a built in "clearfix" so it does not require a clearing element at the end. It also handles centering the grid layout within the page. By default, it has a max width of 940px.

Here is an example of a simple use case. 

<div class="w-container">
  <div class="w-col-3">
    I am 33.3% wide.
  </div>
  <div class="w-col-2">
    I am 50% wide.
  </div>
  <div class="w-col-4">
    I am 16.6% wide.
  </div>
</div>

How to use the contact form

This template comes with a fully working contact form. The base version which will be delivered with this theme supports 4 different fields:

  • Name
  • Email
  • Subject
  • Message


by default all the fields are mandatory. If you need more field or just change the message outputs please open the contact.php and change the settings part to fit your needs. After you have changed these values you are already able to send emails. 

$to_Email = "denibozo1990@gmail.com"; //Replace with recipient email address

Need more fields? Here's an example:

At first of all we are making the change to the html form. In our case we would like to add an optional set of checkboxes. The Basic Markup would look like this: 

<form id="email-form" name="email-form" method="POST" data-name="Email Form">
    <input class="w-input text-field" id="name" type="text" placeholder="Enter your name" 
      name="name" data-name="Name" required>
    <input class="w-input text-field" id="email" type="email" 
      name="email" placeholder="Enter your email address" data-name="Email" required>
     <input class="w-input text-field" id="subject" type="text" 
      name="subject" placeholder="Enter your subject" data-name="Subject">
     <textarea class="w-input text-area" id="text-area" 
      name="message" required placeholder="Your message here..." data-name="Text Area"></textarea>
     <button class="w-button submit-button" type="submit">Submit Message</button>
</form>

As you can see , we are using some kind of special name for these input fields "Howdidyoufindus[]". This technique makes sure we are storing all data inside an array when submitting it.

Now we open the contact.php and adding a new code which is able to display the entered data inside the array properly. Now we need to place a loop which can process this array. We recommend to place this code after line 57 

        
// Phone
$message.= '
 Phone : ' . $_POST['phone'] . '
';

// Checkbox 
if(isset($_POST['Howdidyoufindus']) && is_array($_POST['Howdidyoufindus']) ) {
    
    $message.= '<p>How did you find us? : <br>';
    
    foreach($_POST['Howdidyoufindus'] as $element) {
        $message.= '-' . $element . "<br>";
    }
    
    $message.= '</p>';

}

With the help of the foreach loop we can now concatenate all checked boxes and make them appear inside the message body.
